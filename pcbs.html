Learning Kicad

Here's a rough summary of how to use Kicad.

  1. (If you're using a laptop with a trackpad, plug in a real mouse.)
  2. Make a new project.
  3. Add components to the schematic file.
  4. Connect the components together in the schematic.
  5. Associate a footprint with each component.
  6. Generate a netlist from the schematic.
  7. Read the netlist into the PCB file.
  8. Make a mechanical outline of the PCB on the Edge.Cuts layer.
  9. Roughly arrange the parts the way you want them.
  10. Place mechanically critical components.
  11. Place the rest of the components and route traces between everything.
  12. Run a design rule check (DRC).
  13. Repeat cycle: schematic/footprints -> gen. netlist -> read netlist -> tweak layout -> run DRC
  14. Submit .kicad_pcb file to OSH Park (or possibly generate Gerber files for another manufacturer).
